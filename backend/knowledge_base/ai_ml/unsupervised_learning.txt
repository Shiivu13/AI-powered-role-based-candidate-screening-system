Chapter: Unsupervised Learning — Clustering and Dimensionality Reduction

Unsupervised learning finds structure in unlabeled data. There is no target variable to predict; the goal is to discover inherent patterns, groups, or representations.

K-MEANS CLUSTERING
K-means partitions n observations into k clusters where each observation belongs to the cluster with the nearest mean (centroid).

Algorithm (Lloyd's algorithm):
1. Initialize k centroids (random, or K-means++ for better initialization)
2. Assign each point to nearest centroid
3. Update centroids to mean of assigned points
4. Repeat steps 2-3 until convergence

Objective: minimize within-cluster sum of squares (WCSS) = Σᵢ Σ_{x∈Cᵢ} ||x - μᵢ||²

Limitations: assumes spherical clusters, sensitive to outliers, requires specifying k, can converge to local minima. Multiple random restarts mitigate local minima.

Choosing k: Elbow method (plot WCSS vs k, look for elbow), Silhouette score (measures how similar a point is to its own cluster vs others, range [-1, 1]).

HIERARCHICAL CLUSTERING
Builds a hierarchy of clusters without requiring k upfront.
- Agglomerative (bottom-up): start with each point as its own cluster, repeatedly merge closest clusters
- Divisive (top-down): start with one cluster, recursively split

Linkage criteria define distance between clusters:
- Single linkage: minimum distance between any two points in different clusters (can produce elongated clusters)
- Complete linkage: maximum distance (produces compact clusters)
- Average linkage: average of all pairwise distances (balanced approach)
- Ward's linkage: minimize increase in total within-cluster variance (often best for spherical clusters)

Dendrogram: tree diagram showing the hierarchy; cut at any height to get clusters.

DBSCAN (Density-Based Spatial Clustering of Applications with Noise)
Groups points in high-density regions, marks low-density points as outliers.
Parameters: ε (radius), MinPts (minimum points to form dense region)
- Core points: have ≥ MinPts neighbors within ε
- Border points: within ε of a core point but fewer neighbors
- Noise: not reachable from any core point

Advantages: discovers arbitrary-shaped clusters, robust to outliers, doesn't require k. Disadvantage: sensitive to ε and MinPts, struggles with varying densities.

PRINCIPAL COMPONENT ANALYSIS (PCA)
PCA finds the directions of maximum variance in data (principal components) and projects data onto a lower-dimensional subspace.

Steps:
1. Center data (subtract mean)
2. Compute covariance matrix: C = (1/n) X^T X
3. Compute eigenvalue decomposition: C = VΛV^T
4. Sort eigenvectors by eigenvalue (descending)
5. Project data: Z = XV_k (top-k eigenvectors)

Interpretation: first PC explains most variance, each subsequent PC is orthogonal and explains remaining variance. Explained variance ratio guides choice of components. PCA is a linear method; it finds linear combinations of features.

Applications: dimensionality reduction for visualization, noise reduction, feature extraction before classification, addressing multicollinearity.

t-SNE (t-Distributed Stochastic Neighbor Embedding)
Nonlinear dimensionality reduction for visualization. Converts high-dimensional pairwise distances to probabilities, then finds low-dimensional embedding that preserves these probabilities. Excellent for exploring cluster structure in high-dimensional data (e.g., word embeddings, single-cell RNA). Not suitable for downstream tasks (non-deterministic, slow for large datasets). UMAP is a faster, more scalable alternative.

AUTOENCODERS
Neural network-based dimensionality reduction:
- Encoder: compresses input to low-dimensional latent representation z
- Decoder: reconstructs input from z
- Trained to minimize reconstruction error ||x - decoder(encoder(x))||²

The bottleneck latent space z captures the most important structure. Variational Autoencoders (VAEs) learn a latent distribution, enabling generative sampling.

GAUSSIAN MIXTURE MODELS (GMMs)
Probabilistic model: data is generated by a mixture of K Gaussian distributions. Learns via Expectation-Maximization (EM algorithm). Soft assignment — each point has probability of belonging to each cluster. More flexible than k-means (allows elliptical clusters).

ANOMALY DETECTION
Unsupervised approaches: Isolation Forest (randomly partitions space, anomalies are isolated quickly), Local Outlier Factor (LOF), One-class SVM. Statistical approaches: fit distribution, flag low-probability points.
